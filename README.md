# Classification-of-tree-species-from-images-of-leaves
This project's goal is to create a classification algorithm to train a model to be able to determine the species of a tree using images of its leaves. While this problem concerns image classification, there is a secondary concern of how to work with and analyze an exceedingly large data set. I addressed this problem using the power method and minimization functions to compute the dominant eigenvalues of the images after converting them into matrices of their black and white color vectors. Then, I computed the distances of the eigenvectors' orthogonal projections and utilized their distances to identify which set of tree images they belong to so as to determine their species. Afterwards, I simulated the results, created images and confusion charts to measure my results and conducted convergence analysis to measure the veracity of my results.
